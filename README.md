# Donut Guild Resources (Datapack / Resource Pack)

Donut Guild Resources is a data pack designed with the intention of adding fun and custom content for a Minecraft server as seen on the GuildSMP Season 3. Featuring over a dozen donuts with special effects, new custom items and more!
